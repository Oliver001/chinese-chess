# 中国象棋 Chinese Chess

Java 实现的中国象棋，内置高强度 AI 引擎。

## 功能特性

- **AI 引擎**：Alpha-Beta 剪枝 + 置换表（TT）+ PVS + 空着剪枝 + 历史启发 + 杀手走法
- **开局选边**：支持执红/执黑，AI 难度可调
- **时制**：加秒制 / 包干制 / 无限制，9 种预设（全国联赛、正规慢棋、超快棋等）
- **棋盘翻转**：人方始终在下方
- **打谱复盘**：自动保存对局，支持逐步回放 + 优势折线图
- **视觉高亮**：上一步高亮、选中棋子高亮、可走位置提示
- **音效**：走棋/将军/胜负音效

## 运行方式

```bash
java -jar chess_v14.jar
```

需要 JDK 8+。

## 编译

```bash
# Maven（推荐）
mvn clean package
# 输出：chess_v14.jar
```

## AI 引擎关键修复（v14）

- **置换表归一化方向错误**：正绝杀存入时应 `+ply`，读出时 `-ply`，方向反了导致大量TT误命中，是AI"送子"的根本原因
- **空着剪枝 Zobrist key 错误**：空着后应传对方视角 key（`recomputeZobrist(board, !maxing)`）
- **PVS 最小化节点窄窗口错误**：最小化节点应用 `(beta-1, beta)` 而非 `(alpha, alpha+1)`
- **空着剪枝框架错误**：绝对视角 minimax 不使用负极大值，截断条件需按 maxing 方向判断
